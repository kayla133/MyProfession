# Project Title To-Do List Manager

This program will allow the user to create a to-do list and manage what they have accomplished and what they still need to work on. The user will also be able to save and load their list to a file.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Understand simple C++ functions and operators.
2. Layout code using comments.
3. Create functions that allow the user to interact with the program.

Instructions for using the software:

1. Run the program.
2. Choose from the displayed menu option.
3. Follow the prompts given to the user.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Download Visual Studio
* Download the C++ library
* Build To-Do application
* Run application
